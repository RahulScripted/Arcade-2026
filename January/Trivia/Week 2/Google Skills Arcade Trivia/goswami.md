# Google Cloud Trivia - Week 2

## Question 1: Multiple Network Interfaces
**You are setting up a secure appliance in Google Cloud that needs to inspect traffic from two separate VPC networks. How will you connect this single VM to both networks?**

Select ONE answer that would be relevant:
- A) Use Google Cloud Storage
- B) Create two separate VMs and connect them with a cable
- C) Configure the VM with multiple Network Interfaces (nic0, nic1)
- D) Use a larger hard drive

**Answer: C) Configure the VM with multiple Network Interfaces (nic0, nic1)**

## Question 2: Check Network Interface IPs
**You have a VM with two network interfaces in Google Cloud. How will you check the internal IP addresses assigned to each interface?**

Select ONE answer that would be relevant:
- A) Restart the VM
- B) Use Cloud SQL
- C) Run the command ip addr or check the Google Cloud Console
- D) Create a Firewall Rule

**Answer: C) Run the command ip addr or check the Google Cloud Console**

## Question 3: gcloud Help Command
**You are building a global travel app on Google Cloud and want to translate hotel descriptions from English to Japanese. How will you achieve this?**

Select ONE answer that would be relevant:
- A) Run the command kubectl get pods
- B) Run the command ls -la
- C) Run the command gcloud help
- D) Run the command docker ps

**Answer: A) Run the command kubectl get pods**

## Question 4: Expose GKE Application
**Your GKE application on Google Cloud needs to be accessible to the public internet. How will you expose it to external traffic?**

Select ONE answer that would be relevant:
- A) Use a private IP address
- B) Create a VPN tunnel
- C) Create a Service of type LoadBalancer
- D) Share your password

**Answer: C) Create a Service of type LoadBalancer**

## Question 5: Build Docker Image
**You have written a "Dockerfile" for your application in Google Cloud Shell. How will you turn this into a runnable image?**

Select ONE answer that would be relevant:
- A) Run the command git commit
- B) Run the command gcloud compute instances create
- C) Run the command docker build -t my-app
- D) Run the command mkdir new-folder

**Answer: C) Run the command docker build -t my-app**

## Question 6: List Docker Images
**You want to see which container images are currently stored locally on your Google Cloud Shell machine. How will you list them?**

Select ONE answer that would be relevant:
- A) Run the command ls -la
- B) Run the command gcloud help
- C) Run the command docker images
- D) Run the command docker run

**Answer: C) Run the command docker images**
