# Google Cloud Trivia - Week 1

## Question 1: Audio Transcription
**You have a long audio recording of a conference stored in Google Cloud Storage. How will you transcribe this audio into text using Google Cloud?**

Select ONE answer that would be relevant:
- A) Listen to it and type it out manually
- B) Use the Cloud Translation API
- C) Use the Cloud Speech-to-Text API longRunningRecognize method
- D) Use Google Docs voice typing

**Answer: C) Use the Cloud Speech-to-Text API longRunningRecognize method**

## Question 2: Profanity Filter
**You want to filter out inappropriate words from the transcripts generated by your application automatically. How will you configure the Cloud Speech API to do this?**

Select ONE answer that would be relevant:
- A) Use a firewall rule
- B) Delete the words manually after transcription
- C) Use Cloud Vision
- D) Set the profanityFilter field to true in your request configuration

**Answer: D) Set the profanityFilter field to true in your request configuration**

## Question 3: Translation Service
**You are building a global travel app on Google Cloud and want to translate hotel descriptions from English to Japanese. How will you achieve this?**

Select ONE answer that would be relevant:
- A) Use the Cloud Speech API
- B) Send a request to the Cloud Translation API
- C) Use a paper dictionary
- D) Use Cloud Bigtable

**Answer: B) Send a request to the Cloud Translation API**

## Question 4: Language Detection
**You have a list of user comments in different languages stored in Google Cloud. How will you identify which language each comment is written in?**

Select ONE answer that would be relevant:
- A) Sort them alphabetically
- B) Guess based on the name
- C) Call the detect endpoint of the Cloud Translation API
- D) Use Cloud Storage

**Answer: C) Call the detect endpoint of the Cloud Translation API**

## Question 5: Dataprep Scheduling
**You have built a Dataprep flow to clean your daily sales logs. How will you schedule this flow to run automatically every night at 3 AM?**

Select ONE answer that would be relevant:
- A) Use Cloud VPN
- B) Wake up at 3 AM and click run
- C) Use a stopwatch
- D) Add a Schedule to the Flow output in Dataprep

**Answer: D) Add a Schedule to the Flow output in Dataprep**

## Question 6: Recipe Execution
**You have finished creating a "Recipe" of cleaning steps in Cloud Dataprep. How will you execute these steps on your entire dataset in Google Cloud?**

Select ONE answer that would be relevant:
- A) Create a new Virtual Machine
- B) Click Delete
- C) Send it to a printer
- D) Click Run Job to process the data with Dataflow

**Answer: D) Click Run Job to process the data with Dataflow**
